# QR Code Generator Pro

A modern, feature-rich QR code generator web application built with Python (Flask) and a responsive HTML/CSS/JS frontend. Create customized QR codes with colors, styles, and logos.

## ✨ Features

- **Multiple QR Styles**
  - Basic – custom fill & background colors
  - Styled – rounded corners or gradient effects
  - With Logo – overlay your brand logo (high error correction)

- **User‑friendly Interface**
  - Real‑time color preview
  - Drag & drop logo upload (optional)
  - Responsive design (mobile/tablet/desktop)

- **Download & Share**
  - One‑click download as PNG
  - Preview before saving

- **Helpful Tips**
  - Best practices for scannability
  - Minimum size & margin recommendations
